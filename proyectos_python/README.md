1. Conversor de monedas

   ```python
    pesos = input("Cuanto pesos Argentino tienes ?" )
    pesos = float(pesos)
    valor_dolar = 200
    dolares = pesos / valor_dolar
    dolares = round(dolares, 2)
    dolares = str(dolares)
    print("Tienes $" + dolares + "dolares")
   ```
